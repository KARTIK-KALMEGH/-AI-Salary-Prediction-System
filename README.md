# AI Salary Prediction System

This project predicts employee expected salary using Machine Learning.

## Features

* Salary prediction using Random Forest ML model
* Streamlit web application
* Data preprocessing and training notebook
* Fair salary prediction to reduce bias

## How to Run Project

### 1. Install libraries

pip install pandas numpy scikit-learn streamlit joblib

### 2. Train model

Go to notebook folder:
python salary_prediction.py

### 3. Run website

Go to app folder:
streamlit run app.py
